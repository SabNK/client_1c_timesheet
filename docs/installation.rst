============
Installation
============


Stable release
--------------

To install Client 1C for Time Sheet, run this command in your terminal:

.. code-block:: console

    $ pip install client_1c_timesheet

This is the preferred method to install Client 1C for Time Sheet, as it will always install the most recent stable release.

If you don't have `pip`_ installed, this `Python installation guide`_ can guide
you through the process.

.. _pip: https://pip.pypa.io
.. _Python installation guide: http://docs.python-guide.org/en/latest/starting/installation/


From sources
------------

The sources for Client 1C for Time Sheet can be downloaded from the `Github repo`_.

You can either clone the public repository:

.. code-block:: console

    $ git clone git://github.com/SabNK/client_1c_timesheet

Or download the `tarball`_:

.. code-block:: console

    $ curl -OJL https://github.com/SabNK/client_1c_timesheet/tarball/master

Once you have a copy of the source, you can install it with:

.. code-block:: console

    $ python setup.py install


.. _Github repo: https://github.com/SabNK/client_1c_timesheet
.. _tarball: https://github.com/SabNK/client_1c_timesheet/tarball/master
